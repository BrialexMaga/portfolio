NAME

Cinemadd - Cinema Booking Simulator

SYNOPSIS

A Django based demo application that models a real cinema booking flow, from viewing available screenings to selecting and reserving seats. It integrates PostgreSQL for data handling and Django Testing Framework to ensure code quality and reduce future errors.
